# Stacked cards

Stacked cards animation inspired by Apple Music website, I used Gsap 3 and ScrollScene which is an extra layer on top of ScrollMagic

## Building and running on localhost

First install dependencies:

```sh
npm install
```

Start dev

```sh
npm start
```

Build

```sh
npm run build
```
